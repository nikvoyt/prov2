C++ Это язык программирования, который был разработан в 80-х годах прошлого века как расширение языка C

Что такое программирование?
Программирование — это процесс создания компьютерных программ с помощью написания кода на специальных языках программирования. Оно включает в себя разработку алгоритмов, написание инструкций для компьютера и тестирование программ, чтобы они выполняли определённые задачи.

Vue.js

 Это прогрессивный JavaScript-фреймворк для создания пользовательских интерфейсов (UI) и одностраничных приложений (SPA, Single-Page Applications). Он разработан для упрощения разработки веб-приложений за счёт своей гибкости, простоты интеграции и высокой производительности.
React Это популярная JavaScript-библиотека с открытым исходным кодом, разработанная компанией Facebook (ныне Meta) для создания пользовательских интерфейсов (UI)!
